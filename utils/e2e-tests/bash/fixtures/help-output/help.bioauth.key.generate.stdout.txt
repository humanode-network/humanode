Generate the bioauth key

Usage: humanode-peer bioauth key generate [OPTIONS]

Options:
      --words <WORDS>
          The number of words in the phrase to generate. One of 12 (default), 15, 18, 21 and 24

      --output-type <FORMAT>
          output format
          
          [default: text]

          Possible values:
          - json: Output as json
          - text: Output as text

  -h, --help
          Print help (see a summary with '-h')

  -V, --version
          Print version
