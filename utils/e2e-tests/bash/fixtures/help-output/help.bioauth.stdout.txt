Biometric authentication related subcommands

Usage: humanode-peer bioauth <COMMAND>

Commands:
  key           Bioauth key utilities
  auth-url      Web App URL with bound RPC URL
  api-versions  API versions print
  help          Print this message or the help of the given subcommand(s)

Options:
  -h, --help     Print help
  -V, --version  Print version
