Inspect the bioauth key

Usage: humanode-peer bioauth key inspect [OPTIONS] --suri <SURI>

Options:
  -m, --suri <SURI>
          The secret key uri (mnemonic)

      --output-type <FORMAT>
          output format
          
          [default: text]

          Possible values:
          - json: Output as json
          - text: Output as text

  -h, --help
          Print help (see a summary with '-h')

  -V, --version
          Print version
